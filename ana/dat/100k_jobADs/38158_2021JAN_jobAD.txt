 Freelance web developer LOCALCONTRACT 
 Job Description 
 Are there any UK Berkshire based (or close by) developers interested in some on-site freelance work? 
 We are on the lookout for a highly motivated and talented front-end developer to freelance at our fantastic growing agency. Working with our senior web developer you will work together in the production of Avatar’s digital output. 
 The role: 
 • To work onsite on a range of Craft CMS builds, in conjunction with other internal and external team members. 
 • Creating the HTML, CSS, Javascript files from supplied Sketch and Invision files. 
 • Bring the code into Craft CMS, setting up the CMS fields, sections, structures etc and building out the Craft template files etc. 
 • To use best practice approaches to create well structured, maintainable and documented code. 
 The candidate will have: 
 • A solid background in front-end development, working for a minimum of 4 years in a digital/design agency multi-client environment. 
 • Experience in liaising with clients and external partners when required. 
 • The ability to pick up projects mid-development 
 • To add value with ideas and future proofing 
 Skills: 
 • Craft CMS 
 • Twig, HTML, Sass, CSS, Javascript 
 • Semantic markup 
 • Responsive 
 • Git Version control 
 • Gulp 
 • Bootstrap 
 • Optimisation 
 • Accessibility 
 • Security
