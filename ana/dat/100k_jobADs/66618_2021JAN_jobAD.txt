 Freelance Craft Developer REMOTECONTRACT 
 Job Description 
 We are looking for a web developer to work with us on Craft CMS-powered website projects, on an ongoing freelance basis. If you are confident using modern front end web technologies and PHP, and have built performant Craft 3 sites, we would love to hear from you. 
 Desirable skills/experience: 
 
 Git version control 
 Gulp/Grunt 
 Sass/Susy 
 Javascript 
 
 Your role would be to provide additional support to Dynam’s web team, maintaining existing Craft websites and building new ones. 
 Dynam is a successful marketing, branding and web agency with offices in Inverness and Glasgow, Scotland. Remote working is an option so we welcome applications from anywhere in the UK.
