 Gleeson Recruitment Group are exclusively recruiting a UX Designer for a Midlands based business. 
 The UX Designer position will encompass: 
 
 Conduct user research and evaluate user feedback 
 Conduct concept and usability testing 
 Create wireframes, storyboards, user flows, process flows and site maps 
 Collaborate with Digital management and Dev to define and implement solutions for the website direction, visuals and experience. 
 
 The ideal candidate for this UX Designer role will possess the following skills/expertise - 
 
 Solid experience in creating wireframes, storyboards, user flows, process flows and site maps 
 Proficiency in Photoshop, Illustrator and wire-framing tools 
 Excellent visual design skills 
 Demonstrable track record developing and delivering exceptional user journeys. 
 
 This is a permanent position. 
 Highly competitive package.
