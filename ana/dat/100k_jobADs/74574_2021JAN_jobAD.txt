 Adzuna Job Importer 
 Posted on  November 17, 2014  by  socialcms 
 Adzuna Job Importer Plugin Import job from Adzuna API according to your given parameter. 
 Features  : 
 
 Import job from Adzuna API. 
 Custom import according to your parameter. 
 Set Multiple Campaign/Importer. 
 Makes auto-blogging. 
 Advance keyword search option 
 Support WPMU. 
 Custom template option. 
 Set cron option. 
 Multiple Adzuna API account data import. 
 
 More Detail :  https://socialcms.wordpress.com/contact-us/ 
 
 
 
 Install Adzuna Jobs Importer WordPress Plugin 
 
 
 * Upload the Adzuna Job Importer WordPress Plugin folder to the /wp-content/plugins/ directory
* Activate the Adzuna Job Importer WordPress Plugin through the ‘Plugins’ menu in WordPress
* Go **Adzuna Importer** in admin menu and add new importer search parameter click save button
* In importer list click **Fetch Now** link.
 
 
 
 Plugin Help File 
 
 
 After installing Adzuna Job Importer WordPress Plugin, click on –  Add New . The products are imported from Adzuna Database to WordPress Blog Post. 
 * #### Adzuna Job Importer Settings

  *         * **Campaign Name**  
 
 Run multiple campaigns like “PHP”,“IT“ etc with multiple Adzuna Accounts and fetch products from Adzuna .this is only for differentiate importer.
*  App ID 
Your App ID from Adzuna API. Don't you have such a key?  Request one here 
*  App Key 
Your App Key from Adzuna API. Don't you have such a key?  Request one here 
* **Keyword 
**The keywords to search for. Use space or comma characters to separate multiple keywords
* **Location 
**Free-text matching against company location names or ID from API Adzuna. The geographic center of the search. Place names or postal codes may be use
* **Job Company 
**Free-text matching against company name or ID from API Adzuna
* **Job Category 
**Import this category jobs
* **Job Type 
**Import this job type jobs
* **Max Items to Import 
**Maximum value is 20; we recommend that you set the Max Item Import parameter to 10
* **Feed Status 
**The products will be auto fetched if the feed status is active
* #### WordPress Settings 
   *         * **New Post Status**  
 
 The products can be directly published in the blog or stored in draft section for approval at a later stage
*  Category Name 
List of Categories from WordPress Blog. The Adzuna job imported from above settings will be inserted in that category
*  Run Every 
Built in cron feature that automatically fetches products from Adzuna site that can be set to run after specific periods like day, week etc
*  Display Template 
List of fields that will be displayed in the product description page like job_company job_location, job_description etc.
