 WP Job Manager CareerBuilder Job Importer 
 Posted on  May 31, 2014  by  socialcms 
 WP Job Manager CareerBuilder Job Importer Plugin Import job from CareerBuilder according to your given parameter. This is WP Job Manager Add-ons 
 Features  : 
 
 Import job from CareerBuilder . 
 Advance keyword search option. 
 Set Multiple Campaign/Importer 
 Custom import according to your parameter. 
 Makes auto-blogging. 
 Support WPMU. 
 Custom template option. 
 Set cron option. 
 Multiple CareerBuilder account data import. 
 
 More Detail :  https://socialcms.wordpress.com/contact-us/   
 
 
 Install WP Job Manager CareerBuilder Job Importer WordPress Plugin 
 
 
 * Upload the WP Job Manager CareerBuilder Job Importer WordPress Plugin folder to the /wp-content/plugins/ directory
* Activate the WP Job Manager Indeed Job Importer WordPress Plugin through the ‘Plugins’ menu in WordPress
* Go **JM** **CareerBuilder Importer** in admin menu and add new importer search parameter click save button
* In importer list click **Featch Now** link.
 
 
 
 Plugin Help File 
 
 
 After installing WP Job Manager CareerBuilder Job Importer WordPress Plugin, click on –  Add New . The products are imported from CareerBuilder Database to WP Job Manager job Listing. 
 * #### CareerBuilder Job Importer Settings

  *         * **Campaign Name** Run multiple campaigns like “PHP”,”IT” etc with multiple CareerBuilder Accounts and fetch products from Indeed.this is only for differnciate importer.
    * **Developer Key** Your DeveloperKey from api.careerbuilder.com. Don't you have such a key? [Request one here](http://api.careerbuilder.com/RequestDevKey.aspx)
    * **Keyword** Import this keyword base
    * **Country** Import this County jobs
    * **Location** Import this Location jobs
    * **Category** Import this category jobs
    * **Job Type** Import this employee type jobs
    * **Max Items to Import** Maximum value is 20; we recommend that you set the Max Item Import parameter to 10
    * **Feed Status** The products will be auto fetched if the feed status is active
* #### WP Job Manager Settings

  *         * **New Post Status** The products can be directly published in the blog or stored in draft section for approval at a later stage
    * **Category Name** List of Categories from WP Job Manager . The Indeed job imported from above settings will be inserted in that category
    * **Job Type** List of job Types from WP Job Manager . The Indeed job imported from above settings will be inserted in that category
    * **Job Location** Use Macro {job_location},{job_location_full} or Your fix location
    * **Job Company** Use Macro {job_company} or Your given company.
    * **Application email/URL** Use Macro {job_detail_url} or Your given url or e-mail address.
    * **Job Expires** job Expires period
    * **Run Every** Built in cron feature that automatically fetches products from Indeed site that can be set to run after specific periods like day, week etc
    * **Display Template** List of fields that will be displayed in the product description page like job_company job_loction , job_description etc.
