 About the project : 
 It is a platform that manages cloud security and automatically manages cloud security. 
It allows developers and DevOps to deliver a cloud product with a lease-privileged security configuration. The company's platform creates a customized security configuration for the cloud environment, allowing the R&D and Ops teams to have a secured infrastructure. 
 Requirements : 
 
 4+ years experience with Front-end development 
 Proficient knowledge of VueJS or React framework 
 Experience in working with UX and using Zeppelin (or similar tools) is a plus 
 Knowledge in D3 is a plus 
 Knowledge in AWS (or other cloud vendors) is a plus 
 
 Expectations : 
 
 Should be able to build a project from scratch 
 Currently they have an MVP written in VueJS but don't mind to rewrite it in React 
 The scope of the project is to implement the entire frontend using the mockups which were created using Zeppelin
