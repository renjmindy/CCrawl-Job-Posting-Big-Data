 use 
 Cookie from PHP (programming language), PHP data identifier. Contains only a reference to the current session. No information is stored in the user's browser and this cookie can only be used by the current website. This cookie is mainly used in forms to improve the user experience. Data entered in forms is stored for a short time, for example, if there is an input error by the user and the user receives an error message. Otherwise all data would have to be re-entered.
