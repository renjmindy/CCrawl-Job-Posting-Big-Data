 Position Summary 
 The IT support employee will respond to inquiries from users and diagnose and resolve problems. Additionally they will be responsible for installing, maintaining, and upgrading desktop workstations. 
 Duties and Responsibilities: 
Install, modify, clean, and repair computer hardware 
Install, modify, and maintain computer software 
Insure that all printers throughout the resort are functioning 
Research pricing options for different hardware components 
Backup for Network Administrator including server hardware, server software and operating systems, network infrastructure and active directory 
Investigate any hardware/software problems 
Create/maintain standard operating procedures (SOP’s) 
Maintain desktop operating systems 
 Qualifications: 
Able to work well with minimal supervision 
Experience with computers both hardware and software 
Experience with servers and network infrastructures
