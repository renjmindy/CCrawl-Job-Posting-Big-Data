 JobEngine WordPress Theme - Engine Themes 
 December 27, 2012  Mike Peter  In  Engine Themes ,  JobBoard Theme ,  Theme Type 2370 
 JobEngine Theme  Review 
 JobEngine WordPress Theme  by  Engine Themes  is job board theme developed with super advanced functionality.  JobEngine Theme  can ease down the task of various jobs listing using custom job board functionality. The template provides useful customization tools along with simple user interface so even general user can manage whole platform easily. 
 JobEngine WordPress theme  is developed with features, the first most useful and unique feature job search box with Ajax Filters which means it will automatically filter the job search result as you type without reloading the page. Homepage area can feature pending jobs and latest jobs where you will get a button to load more new jobs so you won't have to go through numeric page navigation. 
 JobEngine Theme  has special job posting area where employer can post their jobs and you can define free and premium plans in this section. The template comes with different payment options like 2Checkout via an API, PayPal, Google Checkout and cash for those who wish to accept payment by direct bank transfer. Payment mode supports various popular currencies like USD, CAD, AUD, CHF, NZD, SGD, HKD. 
 JobEngine WordPress Theme  has special email template to send automated email message to your valuable advertisers and job-seekers. The resolution friendly feature makes it load smoothly with various mobile devices or with any screen size.
