 Using the Workable API to create a careers page

 Updated

 The Workable API will give you access to a list of your open jobs and the details for each role. You can add these to your own careers site, and keep the visual styling directly in line with your companyâ€™s brand guidelines. Every time you create a new position in Workable, your own website will update with the latest details. 
 Read the API documentation to get started 
 Note:  You will need programming/development experience to use the API, or someone from your development team. Workable also offers an option to embed a  job widget  on your website or to use a  Workable-hosted careers page . These types of careers pages can be set up with little to no development experience.
