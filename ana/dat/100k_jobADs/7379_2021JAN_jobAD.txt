 Description: 
 Our client has an immediate need for an  Integration Software Engineer . In this role, you will be responsible for resolving issues prioritized by business partners, through an Agile, cloud-based approach. 
 Requirements: 
 
 3+ years of experience 
 In depth knowledge working with and implementing Microsoft Azure products (Data Factory, Logic App, Micro services, DevOps, CI/CD) 
 Azure Database capabilities:(CosmosDBM Azure SQL DB, Azure SQL Managed Instance) 
 Strong experience in coding languages: (C#, .Net, HTML, CSS, JS) 
 Familiar with implementing polyglot database strategies 
 Microsoft Kusto Query language 
 Strong verbal and written communication skills 
 Experience working in an Agile environment is desirable 
 Capabilities implementing SOAP and RESTful APIs is preferred 
 Prior knowledge of implementation and design of unit tests is useful 
 
 Additional Responsibilities: 
 
 Document works, participate in team meetings, and contribute to project work as needed. 
 Continuously adapt to latest technologies and patterns 
 Collaborate with business and development partners in the creation and implementation of full stack solutions 
 Work together with tech leads to maintain architecture continuity across systems and quality assurance units to test solutions 
 Perform code reviews and resolve incoming production defects 
 Implement deployment processes (CI/CD) 
 Create Proof of Concepts/Prototypes utilizing new technologies to determine technical and cultural fit
