 What you’ll be doing... 
 ** 
 The SDN (Software Defined Networking) Planning group under Verizon Network Technology and Planning organization is looking for a self-motivated and innovative thinking Senior Full Stack Software Engineer in support of Verizon SDN architecture and technology development and deployment. As a senior level member of technical staff, you will be interacting with software engineers, network/system operations staff, network/system architects and vendor partners to provide SDN technology evolution strategies and solutions to keep our service relevance in the market place in the fast moving and quickly evolving networking industry. Proactive technology research, industry trend analysis, and developing next generation network architecture using modern networking technology (e.g., Software Defined Networking) and providing production deployable solutions are key functions of the team. 
 
 Validating new technologies and prototyping. 
 Developing software defined networking (SDN) ecosystems and vendor management. 
 Architecting and developing full-stack software with the focus on applications and data security. 
 Architecting and developing SDN/NFV platforms. 
 Verifying and developing Open API (REST/RESTCONF/NETCONF) and data modeling (YANG). 
 Deploying Linux (KVM, OpenStack, RHEL, CentOS, and Ubuntu) applications and supporting Linux networking lab infrastructure. 
 Participating in open source and open standard industry collaboration activities. 
 
 ** 
 
 What we’re looking for... 
 ** 
 You'll need to have: 
 
 Bachelor’s degree or four or more years of work experience. 
 Six or more years of relevant work experience. 
 Ability to code across the full stack (UI front-end, back-end and database.) 
 Experience with programming languages (Java, C, Go, Python.) 
 Experience with Spring Framework including Spring Boot and NodeJS. 
 Knowledge of Angular, jQuery, ReactJS, Redux, frameworks. 
 
 Even better if you have: 
 
 BS, MS, or PhD degree in Computer Science or EE with modern data communication technology discipline. 
 Five or more years of software development experience. 
 Background working in Linux environments. 
 Knowledge of Angular, jQuery, ReactJS, Redux, Antd frameworks. 
 Open Source tools development, implementation, and/or collaboration experience. 
 Experience developing applications using Microservice architecture. 
 Ability to analyze and implement enhancements to existing applications. 
 Written and verbal communication skills. 
 Experience about general Java web development, understand the concept of Java JSP page, Servlet, Filter, Session, and basic concept of HTTP protocol (method, header, cookie, etc.) 
 Experience about general Java development, understand basic concept like static class, Java collection package, multiple-threading, etc. 
 Experience about general HTML/web development, understand different HTML elements for building web user interface. 
 Experience about general JavaScript development, understand how to make AJAX call, parse JSON data, and make dynamic change to HTML web user interface. 
 Experience about basic database development, capability to write basic SQL and know how to create table, do data query and data change using Java database API (JPA.) 
 Experience using source code control systems (Stash/GIT/GitLab.) 
 Experience in an Agile development environment as part of a team of engineers, developers and testers. 
 Experience with Linux automation/script development (Ansible, Bash.) 
 Experience with Linux virtualization environments/technologies (KVM, OVS, containers) and container scheduling/management platforms (Docker, Kubernetes.) 
 Experience with open source logging and reporting solutions (Logstash, Elasticsearch, Kibana, Grafana, Splunk.) 
 Experience with open source message bus solutions (Kafka.) 
 Database development experience (MongoDB, Redis, and SQL/MySQL/NoSQL.) 
 Open API (REST/RESTCONF/NETCONF) and data modeling (YANG) verification and development. 
 OpenFlow experience. 
 Willingness to travel

 **
